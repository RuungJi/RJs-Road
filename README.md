# RJs-Road
Steam Cities: Skyline Road Assets.

## 도로 라인과 횡단보도의 관계

### Basic 
**A+A** = Segment (S or M or L) / Node SL (ex: 1+1, 3+3, 4+4 ...)

**A+B** = Segment (S or M or L) / Node NSL (ex: 1+2, 3+4, 4+5 ...)

* SL: Stop Line
* NSL: No Stop Line
* B: A+1
